# 🗄 Основы и понятия

## БД, СУБД, РСУБД, SQL

| Понятие | Описание |
|---------|---------|
| **База данных (БД)** | Организованный набор данных. |
| **Система управления базами данных (СУБД)** | Программное обеспечение для работы с БД. |
| **Реляционная СУБД (РСУБД)** | СУБД, использующая таблицы и связи между ними. |
| **SQL** | Язык структурированных запросов для работы с реляционными БД. |

## Первичный и внешний ключ

| Тип ключа | Описание |
|-----------|---------|
| **Первичный ключ (Primary Key)** | Уникальный идентификатор записи в таблице. |
| **Внешний ключ (Foreign Key)** | Поле, ссылающееся на первичный ключ другой таблицы. |

## Типы данных SQL

| Тип данных | Описание |
|------------|---------|
| **INT** | Целые числа. |
| **VARCHAR(n)** | Строка переменной длины. |
| **TEXT** | Длинный текст. |
| **DATE, TIME, DATETIME** | Даты и время. |

## SQL vs NoSQL

| Характеристика | SQL | NoSQL |
|---------------|-----|------|
| **Структура** | Табличная | Документная, графовая, ключ-значение |
| **Гибкость схемы** | Жесткая | Гибкая |
| **Масштабируемость** | Вертикальная | Горизонтальная |

# ⚙️ Индексы, транзакции, триггеры

## Индексы, транзакции, триггеры

| Понятие | Описание |
|---------|---------|
| **Индекс** | Улучшает производительность поиска. |
| **Транзакция** | Группа SQL-операций, выполняющихся как единое целое. |
| **Триггер** | Автоматическое выполнение кода при изменении данных. |

## Хранимые процедуры, функции и представления (VIEW)

| Понятие | Описание |
|---------|---------|
| **Хранимая процедура** | SQL-код, который можно вызывать многократно. |
| **Функция** | Возвращает одно значение и может быть использована в запросах. |
| **VIEW (представление)** | Виртуальная таблица на основе запроса. |

# 🔢 SQL операторы

## DML, DDL, DCL, TCL

| Группа операторов | Описание | Примеры |
|-------------------|---------|---------|
| **DML (Data Manipulation Language)** | Операции с данными | `SELECT`, `INSERT`, `UPDATE`, `DELETE` |
| **DDL (Data Definition Language)** | Определение структуры БД | `CREATE`, `ALTER`, `DROP` |
| **DCL (Data Control Language)** | Управление правами доступа | `GRANT`, `REVOKE` |
| **TCL (Transaction Control Language)** | Управление транзакциями | `COMMIT`, `ROLLBACK`, `SAVEPOINT` |

# 🏗 SQL

## Практика, простые и сложные выборки

### Простые выборки
```sql
SELECT * FROM users;
SELECT name, age FROM employees WHERE age > 30;
```

### Сложные выборки
```sql
SELECT customers.name, orders.amount 
FROM customers 
JOIN orders ON customers.id = orders.customer_id 
WHERE orders.amount > 100;
```
